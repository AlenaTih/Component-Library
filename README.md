# Component Library

## Overview

A library of reusable components built with React and TypeScript.

## Features

Components included:

- **Button**
- **Star**
- **Menu**
- **Badge**
- **Banner**
- **Card**
- **Testimonial**
- **Tooltip**
- **Toast**
